🔍 nmap-automator

Automate nmap scans with smart defaults. Four modes, clean output, no manual flags.

Quick
Top ports, fast scan
Full
All ports, service detection
Vuln
NSE vulnerability scripts
Custom
Your flags, your scan

⚡ Quick start

git clone git@github.com:gnaixnaij/nmap-automator.git
cd nmap-automator
python3 nmap-automator.py <target>

📖 Usage

# Quick scan — top 1000 ports, fast
python3 nmap-automator.py 10.10.10.1 -m Quick

# Full scan — all ports + service detection
python3 nmap-automator.py 10.10.10.1 -m Full

# Vulnerability scan — NSE scripts
python3 nmap-automator.py 10.10.10.1 -m Vuln

# Custom mode — your flags
python3 nmap-automator.py 10.10.10.1 -m Custom -f "-sC -sV -p 80,443"

# Save to file
python3 nmap-automator.py 10.10.10.1 -o output/

📦 Output

  • output/<target>/nmap.gnmap — Grepable format
  • output/<target>/nmap.xml — XML for tools
  • output/<target>/nmap.nmap — Normal output
