Automated nmap scanning

Smart nmap scans,

Automate nmap scans with intelligent defaults. Four modes — Quick, Full, Vulnerability, and Custom — each tuned for the task.

⚡ Quick
Top 1000 ports, fast scan, common services
📡 Full
All 65535 ports with service detection
🛡️ Vuln
NSE vulnerability scripts scan
🔧 Custom
Your flags, your scan, full control

Quick start

git clone git@github.com:gnaixnaij/nmap-automator.git
cd nmap-automator
python3 nmap-automator.py 10.10.10.1 -m Quick

Usage

# Quick scan
python3 nmap-automator.py 10.10.10.1 -m Quick

# Full scan — all ports + service version detection
python3 nmap-automator.py 10.10.10.1 -m Full

# Vulnerability scan — NSE scripts
python3 nmap-automator.py 10.10.10.1 -m Vuln

# Custom mode — your flags
python3 nmap-automator.py 10.10.10.1 -m Custom -f "-sC -sV -p 80,443"

# Save output to directory
python3 nmap-automator.py 10.10.10.1 -o output/

Output

  • output/<target>/nmap.gnmap — Grepable format for parsing
  • output/<target>/nmap.xml — XML for importing into tools
  • output/<target>/nmap.nmap — Normal readable output

Requirements

  • Python 3.6+
  • nmap — Install with sudo apt install nmap
  • Linux, macOS, or Windows (WSL)
